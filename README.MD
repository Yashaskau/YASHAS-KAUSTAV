AgriNova is an AI-powered smart farming platform that provides real-time weather monitoring, climate risk prediction, and sustainable farming recommendations. 
It helps farmers make informed decisions using machine learning, weather APIs, and an intuitive dashboard to improve crop productivity and reduce environmental impact.
🌦️ Real-time Weather Monitoring – Fetches live weather data including temperature, humidity, rainfall, and wind speed.

🤖 AI-Powered Climate Risk Prediction – Detects risks such as heavy rainfall, heat stress, and fungal disease conditions.

🚨 Smart Alerts & Notifications – Provides timely warnings and actionable recommendations to farmers.

🌱 Sustainable Pest Management – Suggests eco-friendly and natural pest-control methods to reduce chemical usage.

💧 Irrigation Decision Support – Helps optimize water usage based on environmental conditions.

📊 Interactive Dashboard – Displays weather data, risk levels, alerts, and recommendations in a simple interface.

📈 Predictive Analytics – Uses machine learning techniques to analyze weather patterns and support crop planning.

🔄 Automated Data Processing – Continuously collects, cleans, and analyzes climate data from external APIs.

🧑‍🌾 Farmer-Friendly Design – Built with a clean and easy-to-understand interface for users with minimal technical knowledge.

⚡ Lightweight & Scalable – Designed to be cost-effective and expandable with future features such as soil analysis, image-based pest detection, and multilingual support.
Tech Stack

Frontend: HTML, CSS, JavaScript

Backend: Python

AI/ML: NumPy, Scikit-learn

API Integration: OpenWeather API

Data Format: JSON

Development Environment: Anaconda / Jupyter Notebook

Version Control: Git & GitHub
